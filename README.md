# Crypto Market Risk Skill

一个原创的 AI Skill 仓库，用于让 AI 以“风险优先、流动性优先、概率优先”的方式分析加密市场、资金费率、事件驱动机会和仓位管理。

## 内容

- `market-risk-arbitrage/SKILL.md`：核心 skill，包含市场分析框架、套利检查清单、仓位纪律和输出格式。
- `market-risk-arbitrage/agents/openai.yaml`：技能列表展示元数据。

## 使用方式

将 `market-risk-arbitrage/SKILL.md` 作为系统提示词或项目指令使用。它不会扮演任何真实人物，也不包含第三方推文存档、头像或未经授权文本。

## 合规说明

本仓库是从零重写的原创版本，仅保留通用交易分析主题：风险控制、流动性、资金费率、事件驱动和复盘纪律。

不构成投资建议。
