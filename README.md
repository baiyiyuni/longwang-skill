# Crypto Market Risk Skill

这是一个面向加密市场分析的原创 AI Persona / Workflow Skill 仓库，聚焦交易风险识别、资金费率与基差套利分析、事件驱动机会判断、仓位纪律和复盘框架。

本项目适合用于把 AI 调整成一种更务实、更风险优先的市场分析助手：先拆收益来源，再拆风险来源，最后给出需要验证的数据和执行边界。

## 仓库内容

1. **market-risk-arbitrage/SKILL.md**：核心 AI 技能文件，封装了风险优先的交易分析框架、套利检查清单、仓位管理纪律和输出格式。
2. **agents/openai.yaml**：Skill 展示元数据，可用于 Codex / OpenAI Skill 列表识别与展示。

## 核心价值

通过本项目，您可以：

- **研究交易员决策框架**：从风险、流动性、费率、基差、事件和仓位纪律等角度拆解市场机会。
- **注入 AI 增强认知**：将 `SKILL.md` 导入支持自定义指令或项目指令的 AI 系统，让 AI 以更清醒的方式分析市场。
- **辅助策略复盘**：围绕入场理由、失效条件、执行价格、退出纪律和实际偏差进行结构化复盘。

## 如何使用本 Skill？

您可以将 `market-risk-arbitrage/SKILL.md` 文件中的内容作为 **System Prompt（系统提示词）** 或项目指令注入到主流 AI 模型中：

1. **直接复制法**：打开 `market-risk-arbitrage/SKILL.md`，复制其中的核心原则、分析流程和输出格式，在对话开始前告诉 AI：“请按以下市场风险分析框架进行思考...”。
2. **Custom Instructions**：在 ChatGPT 的“自定义指令”或 Claude 的“Project Instructions”中粘贴文件内容，使 AI 长期采用该分析框架。
3. **策略咨询**：注入后，您可以询问它：“帮我拆一下这个资金费率套利的收益来源和最大风险。”或“这个代币事件驱动机会应该验证哪些数据？”
